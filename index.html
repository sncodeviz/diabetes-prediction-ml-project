Overview

Project Description

This healthcare analytics–themed project analyzes a real-world clinical dataset to develop a machine learning model that predicts the likelihood of an individual having diabetes based on basic health and demographic features.

Using a structured predictive analytics workflow, the project examines factors such as age, BMI, blood glucose level, and Hemoglobin A1c that contribute to diabetes risk. The project demonstrates the full lifecycle of applied data science, from raw data to actionable insights, while also highlighting the importance of model interpretability, ethical considerations, and responsible machine learning use in health-related analytics.

Findings

Executive Summary

Diabetes is a growing public health concern in the United States and worldwide. Early detection is essential to reduce long-term complications and healthcare costs, particularly given the financial pressures many patients face.

The goal of this project is to build a predictive model that identifies individuals at higher risk of diabetes using basic clinical and demographic features.

The dataset contained over 100,000 patient records, with features including age, gender, BMI, smoking history, blood glucose level, HbA1c level, hypertension, and heart disease status. The target variable indicated whether each patient had been diagnosed with diabetes. The dataset was highly imbalanced (only about 8% positive cases), so SMOTE-based oversampling was used to address class imbalance.

The project followed six phases:

  • Exploratory initial checks and data understanding
  • Data analysis and feature exploration
  • Data cleaning and preprocessing
  • Class imbalance handling with SMOTE
  • Model development (multiple algorithms)
  • Evaluation and interpretation

Three models were evaluated:

  • Logistic Regression (baseline)
  • Decision Tree
  • Random Forest

All models were combined with SMOTE to better represent the minority (diabetic) class.

Performance was compared using precision, recall, F1-score, and overall accuracy. Tree-based models outperformed the linear baseline, especially in identifying diabetic patients. The Decision Tree model achieved the best balance between accuracy and recall for the minority class, making it the preferred option for this dataset.

Feature importance analysis confirmed that blood glucose level and HbA1c level were the strongest predictors, aligning with clinical standards. Age and BMI also contributed meaningfully to risk.

In practice, a model like this could be used as a supportive risk assessment tool to flag patients who may need additional screening, support early intervention, and help prioritize clinical resources.

However, this model is not a diagnostic system and should never replace clinically trained professionals. It is best used to support medical decision-making while respecting patient privacy, fairness, and ethical guidelines.
